About

Glitchy Cyborg is the front end for my personal "super" computer, Gaea. The joke here bing that it made of small ARM architecture quad core single board computers

It's not intended to do any real serious computations but for those looking to develop test or maybe just play around. I'm making it available to others for a couple reason, mostly learning purposes.

Please keep in mind, I'm no expert, just passionately curious

Technical Specs

Gaea consist of a 5 Rock64 single board computers from Pine64 Like most SOCs, the ROCK64 uses ARM cores which use a reduced Instruction (RISC) and are considered more power efficient than their CISC counteparts

Rock64 v2 specs

  • Rockchip RK3328 CPU
    • quad core
    • A53 ARM cores
    • 64 bit
  • 2 Gb DDR3 RAM
  • 32 Gb emmc storage

The cluster uses a Beowolf architecture (1 login node, 4 slave nodes) with 64 Gb of steady state scratch storage. Additional info can be found at Pine64.org
