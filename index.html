About

Glitchy Cyborg is the front end for my personal hobby "super" computer, Gaea. Calling it a supercomputer is of course a joke, but you get the idea. The curent plan being, build this front end, and give people a chance to try MPI on ARM based hobby clusters without having to build one themselves.

I'm making it available to others for a couple reason, mostly learning purposes. Please keep in mind, I'm no expert, just passionately curious.

I'm also considering setting up a forum, where can people can talk about their own projects, get help, post reviews, watever. We'll see.

Of course, this is all very fluid, things can change, not tied to anything

Cheers!

Technical Specs

Gaea consist of a 5 Rock64 single board computers from Pine64 Like most SOCs, the ROCK64 uses ARM cores which use a reduced Instruction (RISC) and are considered more power efficient than their CISC counteparts

Rock64 v2 specs

  • Rockchip RK3328 CPU
    • quad core
    • A53 ARM cores
    • 64 bit
  • 2 Gb DDR3 RAM
  • 32 Gb emmc storage

The cluster uses a Beowolf architecture (1 login node, 4 slave nodes) with 64 Gb of steady state scratch storage. Additional info on other Pin64 products can be found at Pine64.org
